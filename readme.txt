Injects the WordLift Cloud bootstrap script into the head of every page.

== Description ==

WordLift Cloud is a lightweight plugin that connects your WordPress site to WordLift Cloud by injecting the bootstrap script into every page.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
